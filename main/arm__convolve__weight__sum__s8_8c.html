Pre-computes per-output-channel weight sums for a standard convolution.

Parameters
[out]vector_sum_bufPointer to the buffer that will hold the weight sums.
[in]rhsPointer to the filter weights. Data type: int8
[in]input_dimsInput tensor dimensions. Format: [N, H, W, C_IN]
[in]filter_dimsFilter tensor dimensions. Format: [C_OUT, KH, KW, C_IN]
[in]output_dimsOutput tensor dimensions. Format: [N, H, W, C_OUT]
[in]lhs_offsetInput-offset added to every input element before MAC. Range: [-127, 128]
[in]bias_dataOptional bias pointer. Data type: int32
Returns
ARM_CMSIS_NN_ARG_ERROR on invalid arguments, or ARM_CMSIS_NN_SUCCESS on success.
  • Supported framework : TensorFlow Lite Micro
  • The buffer pointed to by vector_sum_buf must be at least
    output_dims->c × sizeof(int32_t) bytes.
