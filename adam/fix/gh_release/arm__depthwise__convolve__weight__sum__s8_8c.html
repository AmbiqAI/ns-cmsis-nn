Pre-computes per-channel weight sums for a depthwise convolution.

Parameters
[out]vector_sum_bufBuffer to hold the computed weight sums.
[in,out]scratch_bufScratch buffer used internally; caller must clear for security.
[in]rhsDepthwise convolution weights. Data type: int8
[in]dw_conv_paramsDepthwise-convolution parameters (stride, dilation, pad, etc.)
[in]input_dimsInput tensor dimensions. Format: [N, H, W, C_IN]
[in]filter_dimsFilter tensor dimensions. Format: [1, KH, KW, C_OUT]
[in]output_dimsOutput tensor dimensions. Format: [N, H, W, C_OUT]
[in]lhs_offsetInput-offset applied before MAC. Range: [-127, 128]
[in]bias_dataOptional bias pointer. Data type: int32
Returns
ARM_CMSIS_NN_ARG_ERROR on invalid arguments, or ARM_CMSIS_NN_SUCCESS on success.
